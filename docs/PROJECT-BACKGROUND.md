# 项目背景与个人目标

## 1. 项目一句话

一个跨平台的私有 TodoList 应用（Windows、macOS、Linux、Android），当前先做 Web MVP。

## 2. 个人起点

1. 后端：目前以 Java 基础为主。
2. 前端：几乎从零开始，需要边做边学 React。
3. 工程能力：希望通过完整项目补齐数据库、并发、网络、部署等认知。
4. Agent 能力：希望系统学习 sub-agent、skills 和高效协作方式。

## 3. 项目目标优先级

1. 学习覆盖面最大化
2. 简历展示深度
3. 上线可用产品

## 4. MVP 必须包含

1. 新建任务、勾选完成、保留完成历史
2. 任务优先级（high/medium/low）
3. DDL 与到期前提醒
4. Token 云同步（不使用账号密码）

## 5. 开发策略

1. 先完成 Web 端 MVP 闭环，再扩展客户端。
2. 一次只做一个小切片，做完即可验证。
3. 文档与代码同步推进，避免上下文漂移。

## 6. Agent 学习目标

1. 能在开发流程中实际使用 Agent（计划、实现、验证、总结）。
2. 能创建并复用 Skill（如 build/test）。
3. 能在不同 Agent 之间做稳定交接（Copilot、Codex 等）。
