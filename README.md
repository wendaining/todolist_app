# Todolist App

跨平台私有 TodoList 项目，当前阶段先完成 Web MVP。

## 当前状态

1. 后端主线：Java 21 + Spring Boot 3
2. 前端主线：React + TypeScript + Vite
3. 当前里程碑：M1 本地单端可用（无同步）
